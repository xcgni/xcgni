# Code of Conduct

## Our standard

Excogni is a welcoming, harassment-free space for everyone, regardless of background or
identity. We expect participants to be respectful, constructive, and kind - the same calm,
honest character the project itself aims for.

Examples of expected behaviour: thoughtful feedback, accepting constructive criticism
gracefully, focusing on what's best for the project and its users, showing empathy.

Unacceptable behaviour includes: harassment, personal attacks, demeaning or discriminatory
comments, publishing others' private information, and other conduct that would reasonably be
considered inappropriate.

## Scope

This applies in all project spaces (issues, pull requests, discussions) and when representing
the project publicly.

## Enforcement

Report unacceptable behaviour via the **in-app feedback form** (the "Feedback" button on any
page; start the message with `[conduct]` and include a way to reach you). It is a private
channel read only by the maintainer. Reports will be
reviewed and handled confidentially. Maintainers may remove, edit, or reject contributions and
participants that violate this code.
